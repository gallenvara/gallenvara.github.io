Metrics in Flink

Flink在1.0版本以后添加了metrics统计系统并可以和其他多种metrics工具结合使用。
目前flink支持四种metrics指标，分别是：

Gauge
Meter
Counter
Histogram

##注册Reporter
定义完metrics之后，我们需要将metrics通过reporter进行展示，flink中默认提供了四种reporter：

JMX(default)
Ganglia
Graphite
StatsD
